# Workout App API

## This project is a REST API built with Express and MongoDB.
## It allows users to create, read, update, and delete workout entries.

## How to run it
1. Run npm install
2. Make sure MongoDB is running
3. Run node server.js
4. Test routes in Postman

## Main Routes
GET /workouts
GET /workouts/:id
POST /workouts
PUT /workouts/:id
DELETE /workouts/:id
